Personal RevealJS Theme

Find code here


In my teaching, I usually find myself needing to teach students both statistics and the R programming language. To this end, I have used Beamer slides in the past, but I have always been unhappy with the lack of customization and the rigidity of the framework.

I found the revealjs framework and the seemless integration with quarto much more flexible for my needs. For example, revealjs, being an html framework, allows to include interactive elements (e.g., shiny apps, interactive 3D plots) right on the slides!

Check out my custom theme that I use for both my presentations and lectures. I have added many features that improve both accessibility and quality of life for both students and presenters! If you like this theme, you are more than welcome to use it!!
